# Drawing Board

A feature-rich web-based drawing application built with HTML, CSS, and JavaScript. Create digital artwork with various drawing tools, colors, and customization options.

## Features

- **Drawing Tools**
  - Brush - Free-form drawing for sketching
  - Line - Draw straight lines
  - Rectangle - Create filled or outlined rectangles
  - Circle - Draw filled or outlined circles
  - Triangle - Draw triangles
  - Eraser - Remove drawn content

- **Customization Options**
  - Color Picker - Select any color for your drawings
  - Fill Color Toggle - Fill shapes with color or keep them outlined
  - Adjustable Brush Size - Slider to control stroke thickness (1-31px)

- **Canvas Management**
  - Clear Canvas - Erase all drawings and start fresh
  - Save Image - Export your artwork as an image file
  - User Name Input - Personalize your drawing sessions

## How to Use

1. **Open the Application**
   - Open `index.html` in your web browser

2. **First Write Your Name then Select a Drawing Tool**
   - Click on any tool from the Shapes section (Brush, Line, Rectangle, Circle, Triangle, or Eraser)
   - The selected tool will highlight in color

3. **Choose Your Color**
   - Click on a color from the Colors section, or use the color picker for custom colors

4. **Adjust Brush Size**
   - Use the size slider to change the thickness of your strokes

5. **Draw on Canvas**
   - Click and drag on the white canvas area to draw with the selected tool

6. **Fill Shapes** (optional)
   - Check the "Fill Colour" checkbox to fill shapes with solid color
   - Uncheck to draw outlined shapes only

7. **Save or Clear**
   - Click **Save** to download your artwork
   - Click **Clear** to erase the entire canvas and start over
