# Fan Controller
An interactive fan simulator with CSS animations and JavaScript controls.

## Features
- Animated rotating fan with 3 speed levels
- Stop button to pause animation
- Speed 1 (2s), Speed 2 (1s), Speed 3 (0.5s per rotation)

## Usage
1. Open `Fan.html` in a browser
2. Click speed buttons (1, 2, 3) to start
3. Click Stop to pause
