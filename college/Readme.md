# SkillSprint

A responsive educational website built with Bootstrap 5, providing coding and academic resources for students.

## Description

i-Coder is a web platform designed to help students access academic materials, study resources, and connect with educational content. The website 
features a clean, modern interface with navigation for different sections including academic sessions, study materials, and contact information.
## Features

- **Responsive Design**: Built with Bootstrap 5 for optimal viewing on all devices
- **Navigation Menu**: Easy access to different sections including Home, About, Study Materials, and Contact
- **Academic Resources**: Links to past year papers, notes, and assignment solutions
- **User Authentication**: Sign-up and login modals for user registration
- **Contact Form**: Contact us page with inquiry forms
- **Academic Information**: Dedicated pages for academic sessions and university information

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Custom styling
- **Bootstrap 5.3.2**: Responsive framework and components
- **JavaScript**: Bootstrap's interactive components

## Project Structure

```
i-coder/
├── index.html          # Main homepage
├── Acedmic.html        # Academic information page
├── Contect.html        # Contact us page         
└── README.md           # Project documentation
```

## How to View

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through the different sections using the menu

No server setup is required as this is a static website.
